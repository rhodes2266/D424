Course Tracker App

This is a mobile app built with .NET MAUI that allows students to track academic terms, courses, and assessments. The app includes notification features for course start/end dates, instructor info, and note-taking. Users can manage terms, view course details, set reminders, and generate reports.

Built for Android devices, this app is useful for students in managing their schedule and coursework.
